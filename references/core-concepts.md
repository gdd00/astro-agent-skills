# 核心概念

### 为什么是 Astro?

Astro 是**以内容驱动的网站**最优的 Web 框架，核心特点：

- **群岛架构（Islands）**：只对需要交互的组件加载 JS，其余部分纯 HTML
- **服务器优先**：默认在服务端渲染 HTML，而非客户端 SPA
- **默认无 JS**：组件默认零 JavaScript，按需添加交互
- **UI 框架无关**：支持 React、Vue、Svelte、Solid、Preact、Web Components 等

### 群岛架构（Islands Architecture）

```
┌─────────────────────────────────┐
│         HTML 页面（静态）         │
│  ┌──────┐       ┌──────┐        │
│  │ Island│       │ Island│       │  ← 仅这些部分加载 JS
│  │ (JS) │       │ (JS) │        │
│  └──────┘       └──────┘        │
└─────────────────────────────────┘
```

- 每个"岛屿"是独立的交互组件
- 其余页面内容保持纯静态，不加载 JS
- 大幅提升首屏加载性能
